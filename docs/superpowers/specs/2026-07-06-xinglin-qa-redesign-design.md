# 杏林问答前端重设计 — 宣纸书卷风

## 概述

将 `static/index.html` 从通用暖棕色调重构为浓郁中国风（宣纸、水墨、印章、古籍）的单文件前端。功能逻辑不变，后端 API 不动，纯视觉与微交互升级。

## 约束

- 单文件 HTML，内联 CSS/JS，零外部框架依赖
- 保持现有所有功能：多会话、流式 WebSocket、历史搜索、暗色模式、快速提问
- 不修改任何后端代码

---

## 色彩体系

| 角色   | 色值       | 用途                       |
| ------ | ---------- | -------------------------- |
| 宣纸底 | `#f5f0e8`  | 页面主背景                 |
| 浓墨   | `#2c2416`  | 正文文字                   |
| 淡墨   | `#8b7a62`  | 辅助文字、meta             |
| 朱砂   | `#c43a31`  | 用户气泡、强调、印章       |
| 赭石   | `#8b5a2b`  | Header 匾额底色            |
| 金泥   | `#b8963e`  | 分割线、边框、icon 点缀    |
| 石绿   | `#5b8c5a`  | 系统回复 meta、成功状态    |
| 花青   | `#3a5a8c`  | 链接、可点击元素次要色     |

## 字体层级

- **标题/匾额**: `STKaiti` / `KaiTi` / `Noto Serif SC`, `letter-spacing: 0.1em`
- **正文**: `Noto Serif SC` → `STSong` → `SimSun` → `serif`
- **代码/数据**: `JetBrains Mono` → `Fira Code` → monospace

## 质感

- 背景：CSS 宣纸纤维纹理 + 四角 radial-gradient 水墨山影
- 卡片/气泡：纸质感（多层 box-shadow 模拟厚度），小 border-radius 仿手工裁纸
- 暗色模式：名为"夜读"

## 组件设计

### Header — 匾额

- 赭石底色，四边细金线边框
- 中央书法大字"杏林问答"，左侧引首章
- 右侧：夜读切换（月亮/灯笼）、GitHub 链接
- 底部光影凹槽感

### 侧边栏 — 古籍目录

- 深色纸背景，线装书纸色
- "历史会话"标题配竖向短线装饰
- 每条记录金线分隔，hover 时左侧朱砂圆点出现
- 搜索框：仿古签条样式，focus 金泥边框
- 底部会话 ID 淡墨色 + 半透明分隔线

### 对话主区域 — 宣纸笺

- 浅色底（`#faf7f0`）+ 四边内阴影暗角
- Welcome 卡片：古人书信格式，竖排装饰线 + 敬语风格问候
- 分类 chips：仿笺条样式

### 聊天气泡

- **用户消息**: 朱砂底白字，方形印章感，border-radius 极小
- **系统回复**: 浅驼色半透明底 + 金线左边框（引文格式）
- 头像: 用户朱砂圆印，系统石绿圆印

### 输入区 — 砚台意象

- 输入框：深灰底仿砚石，白字
- 发送按钮：方形印章，"盖章"hover 效果
- 快速提问按钮：笺条样式

## 动效设计

| 场景             | 效果                                  | 时长     |
| ---------------- | ------------------------------------- | -------- |
| 消息入场         | 下方 8px 淡入 + 上浮                  | 0.3s     |
| 历史条目 hover   | 左侧朱砂圆点 scale 0→1，条目右移 4px  | 0.25s    |
| 按钮 hover       | 上浮 1px + 阴影加深                   | 0.2s     |
| 发送按钮 hover   | "盖章" scale 1→1.05→1                 | 0.25s    |
| 输入框 focus     | 金泥边框 + 洇墨扩散 shadow             | 0.3s     |
| typing 指示器    | 三竖点，毛笔砚台轻点节奏              | 循环     |
| 搜索框 focus     | 金泥微光边框                          | 0.3s     |
| 加载 spinner     | 墨滴扩散 CSS 动画                     | 循环     |
| 欢迎卡片入场     | 子元素 stagger 50ms 依次淡入           | 各 0.3s  |

## 暗色模式 — "夜读"

- 背景: `#1a1c1f` 深蓝灰（深夜书房）
- 宣纸笺: `#222420`（烛光映照的泛黄纸面）
- 文字: `#d4c8b8` 暖灰金色
- 朱砂降低饱和度，金泥增加亮度
- 匾额: 深底 + 微弱金色光晕
- 切换图标: 太阳 → 灯笼

## 实现策略

1. 直接修改 `static/index.html`，保留所有 JS 逻辑不变
2. CSS 变量全部替换为新配色，class 命名保持兼容
3. 删除 `static/src/App.jsx` 和 `static/old_index.html`（废弃文件）
4. 自测流式对话、暗色切换、历史搜索等交互
