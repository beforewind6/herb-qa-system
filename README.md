## 🏥 这是个啥？

这是一个**让 AI 跟你聊中医**的项目。

问它"四君子汤有什么药"，它能秒回；问"足三里在哪"，它能给你精准定位；问"我上火了怎么办"，它会从八纲辨证讲到三焦辨证，然后建议你去找个真医生。

> ⚠️ **免责声明**：本系统不提供医疗建议，只提供知识。有病请去医院，别问 AI。

### 两种模式，双剑合璧

```
你问问题 → FAQ 秒搜（BM25 + MySQL + Redis）
              ↓ 没找到？
           RAG 深度思考（Milvus 向量库 + 大模型生成）
              ↓
           返回一个看起来很像样的答案 ✨
```

| 能力 | 说明 |
|------|------|
| ⚡ **FAQ 快问快答** | BM25 算法检索，命中率 85% 以上，像老中医翻药典一样快 |
| 🧠 **RAG 深度问答** | 向量检索 + Qwen 大模型，不懂的也能给你掰扯半天 |
| 💬 **流式输出** | 一个字一个字往外蹦，像真人打字，仪式感拉满 |
| 🗂 **多轮对话** | 记住你 5 轮对话，不会每次都说"你好我是杏林助手" |

## 🚀 5 分钟跑起来

```bash
# 1. 克隆（自带 21 个高频 TCM 问答数据）
git clone https://github.com/beforewind6/herb-qa-system.git
cd herb-qa-system

# 2. 装依赖
pip install -r requirements.txt

# 3. 🔑 设置 API Key（灵魂所在）
# 编辑 config.ini，把 dashscope_api_key 换成你的百炼 API Key
# 白嫖地址：https://bailian.console.aliyun.com/
# dashscope_api_key = YOUR_DASHSCOPE_API_KEY_HERE

# 4. 确保 MySQL + Redis + Milvus 都活着
# （如果你只改 key 不跑数据库，它会优雅地报错）

# 5. 🔥 启动！
python app.py
```

打开 `http://localhost:8080`，开始调戏 AI 中医。

## 🛠 技术全家桶

| 层级 | 用了啥 | 为什么选它 |
|------|--------|-----------|
| 后端框架 | FastAPI + WebSocket | 快，异步，原生支持 WebSocket |
| FAQ 引擎 | BM25 + MySQL + Redis | 经典算法永不过时，Redis 缓存快到起飞 |
| RAG 引擎 | BGE-M3 + Milvus + Qwen | 中文向量检索顶配，Qwen 中文理解力强 |
| 前端 | 原生 HTML/CSS/JS | 零构建，一个 HTML 打天下，暗色模式自带 |
| 意图识别 | BERT 微调二分类 | 先搞清楚你是来闲聊还是来问专业的 |

## 🎋 杏林的典故

> 三国时期，名医董奉隐居庐山，为人治病不收诊金。只让重病愈者种杏树五株，轻病愈者种一株。数年后，杏树郁然成林，董奉以杏易谷，赈济贫民。
>
> 后世便以**「杏林春暖」**赞誉医者仁心，以**「誉满杏林」**称颂医术高明。

这也是为什么这个项目叫"杏林问答"——我们希望 AI 能成为中医药文化传承的一棵小杏树 🌱

## 🎯 趣味问答挑战

试试问这些问题，看 AI 怎么答：

- "吃枸杞真的能明目吗？" → 肾藏精、肝藏血，肝肾同源……（开始上课）
- "我是不是肾虚？" → 请描述具体症状，不要自我诊断……（AI 很谨慎）
- "中药为什么那么苦？" → 良药苦口利于病，苦味入心……（哲学回答）
- "你能给我开个方子吗？" → 不能，我是 AI 不是医生……（立刻认怂）
